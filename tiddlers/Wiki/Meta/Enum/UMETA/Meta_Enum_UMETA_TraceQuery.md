# TraceQuery

- **功能描述：** Enum /Script/Engine.ECollisionChannel
- **使用位置：** UENUM::UMETA
- **引擎模块：** Enum Property
- **元数据类型：** bool
- **常用程度：** 0

只在ECollisionChannel 上使用，指定哪些通道是用来Trace的。
